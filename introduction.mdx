## What is ForgeUI?

ForgeUI is an **animation-first React component library** that helps you build rich, modern, and interactive user interfaces with minimal setup. From animated forms to shimmer text and motion-enhanced cards, ForgeUI provides production-ready components that emphasize visual delight and micro-interactions.

Perfect for developers who care about UI, detail, and the small touches that make applications feel polished and engaging.

## Technology stack

ForgeUI is built on top of modern, battle-tested technologies:

- **Next.js** - React framework for production
- **TypeScript** - Type safety and better developer experience
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - High-quality component foundation
- **Framer Motion** - Production-ready animation library

## Component highlights

ForgeUI includes 19 animated components focused on visual effects and micro-interactions:

- **Animated Form** - Multi-step forms with elegant animations
- **Text Effects** - Shimmer, reveal, and morph text animations
- **Interactive Cards** - Expandable, fraud detection, and social cards
- **Notification System** - Animated notification center and panels
- **Security Components** - Vault lock, bot detection, and security cards
- **And more** - Animated OTP, tabs, spectrum switcher, and stack ripple

## What makes ForgeUI unique?

Unlike traditional component libraries that add animations as optional features, ForgeUI embraces an **animation-first philosophy**. Every component is designed from the ground up with motion and interaction in mind, creating experiences that feel fluid, responsive, and delightful.

ForgeUI isn't a community-driven project - it's a personal lab for exploring UI, animation, and motion design in React. Everything you see is built with the intent to learn, refine, and share what makes frontend development exciting.
